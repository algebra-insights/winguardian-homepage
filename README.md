# WinGuardian Marketing Webpage

This folder contains the marketing website for WinGuardian.

## Files

- `index.html` - Main webpage (single-page site)

## Screenshots

Place your PNG screenshot files in this directory with the following names:

- `screenshot1.png` - Dashboard Overview
- `screenshot2.png` - File Integrity Monitor
- `screenshot3.png` - AppData Backup (folder tree view)
- `screenshot4.png` - Registry Snapshots

The webpage will automatically display these images when they're present.

## Customization

### Update Download Links

Replace placeholder links in the Download section:
- Change `https://download.winguardian.com/latest/winguardian-setup.exe` to your actual download URL
- Change `https://download.winguardian.com/trial/winguardian-trial.exe` to your trial download URL

### Update Contact Information

Replace placeholder emails and links:
- `sales@winguardian.com` - Sales contact
- `contact@winguardian.com` - General contact
- Update footer links (docs, support, GitHub, etc.)

### Update Pricing

Modify the pricing section if your actual prices differ from the placeholders ($49, $149, Custom).

### Update Version Number

Change "Version 0.2.0" in the download section to match your current release.

## Viewing the Page

Simply open `index.html` in any web browser. The page is fully self-contained with embedded CSS and JavaScript.

## Hosting

To host this webpage:
1. Upload `index.html` and all `screenshot*.png` files to your web server
2. No server-side processing required - pure static HTML
3. Works with any static hosting service (GitHub Pages, Netlify, Vercel, AWS S3, etc.)
